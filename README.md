# testlive
###
使用阿里云直播sdk demo ,支持直播,点播 视频直播与悬浮窗小窗口无缝衔接切换(一般拉流格式 rtmp m3u8等)
###
视频播放基础库 支持全屏,快进,手势基础操作
###
全局视频小窗口 权限判断,高斯模糊背景(可以加深颜色)
###
飘心效果，还有列表单例播放视频 ,列表全屏视频,视频支持手势,进度,亮度,声音
